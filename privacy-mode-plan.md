# 隐私模式与保护内容功能方案

## 目标

在设置中新增公开模式和隐私模式。公开模式下，受保护的文件、文件夹、标签以及相关内容不可见、不可搜索、不可通过任何入口访问。隐私模式下，用户可以查看受保护内容，并设置文件、文件夹、标签的保护状态。

## 模式定义

### 公开模式

公开模式是默认展示模式。该模式下：

- 受保护文件不显示。
- 受保护文件夹不显示。
- 受保护文件夹下的所有子文件和子文件夹不显示。
- 包含受保护标签的文件和文件夹不显示。
- 受保护标签不显示。
- 添加标签、搜索标签、标签自动补全时，不出现受保护标签。
- 搜索结果过滤掉所有受保护内容。
- 不显示保护设置入口。
- 不显示隐藏内容数量。
- 不暴露保护状态或保护内容存在的线索。

### 隐私模式

隐私模式下：

- 可以查看受保护文件、文件夹和标签。
- 可以设置文件、文件夹是否保护。
- 可以在设置中的“标签”内设置标签是否保护。
- 文件和文件夹右键菜单中显示“保护/解除保护”。
- “保护/解除保护”放在“收藏/取消收藏”下面。
- 受保护文件和文件夹显示锁定图标。
- 进入受保护文件夹时，面包屑左侧原本的文件夹图标替换为锁定图标。

## 默认状态

- 默认无隐私密码。
- 默认处于公开模式。
- 首次从公开模式切换到隐私模式时，要求设置隐私密码。
- 设置隐私密码成功后，直接进入隐私模式。

## 密码设置

首次进入隐私模式时弹出密码设置窗口。

规则：

- 使用 HeroUI 的 `InputOTP` 组件。
- 需要输入两次密码进行确认。
- 两次密码一致时，密码设置成功并进入隐私模式。
- 两次密码不一致时，提示设置失败，清空输入框，让用户重试。
- 密码仅允许数字和字母。
- 不允许符号。
- 不允许空密码。

建议：

- 密码长度固定为 6 位或 8 位。
- 如果偏向 PIN 体验，使用 6 位。
- 如果偏向更高安全性，使用 8 位。

## 隐私模式解锁

已设置隐私密码后，从公开模式切换到隐私模式时：

- 弹出密码输入窗口。
- 使用 `InputOTP` 输入密码。
- 密码正确后进入隐私模式。
- 密码错误时提示失败，并允许重试。
- 不显示具体错误原因，不提示哪一位错误。

## 启动时模式恢复

设置中提供开关：

- 名称建议：启动时恢复上次隐私模式。
- 默认关闭。

采用方案 B：

- 如果开关关闭，每次启动默认进入公开模式。
- 如果开关开启，且上次退出时处于隐私模式，则下次启动时显示解锁弹窗。
- 密码验证通过后，恢复进入隐私模式。
- 密码验证失败或取消时，保持公开模式。
- 不直接持久化“已解锁”状态。

## 忘记密码与 Windows 身份验证

忘记隐私密码时，不允许直接解除所有保护标记。

可提供“忘记密码”入口，调用 Windows 系统身份验证。

规则：

- 应用请求 Windows 弹出系统身份验证界面。
- 验证方式由 Windows 决定，可能包括 Windows Hello、PIN、指纹或人脸。
- 应用只接收验证通过或失败的结果。
- 应用不能读取 Windows PIN。
- 应用不能自行校验 Windows PIN。
- Windows 身份验证通过后，允许用户重设隐私密码。
- 重设密码后，原有保护标记保持不变。
- 重设成功后，直接进入隐私模式。
- 如果 Windows 身份验证不可用或验证失败，则不能重设密码，只能继续尝试原隐私密码。

产品文案建议：

> 忘记隐私密码时，可通过 Windows 身份验证重设密码。验证方式由系统决定，可能包括 Windows Hello、PIN、指纹或人脸。

避免使用以下文案：

> 输入 Windows PIN 重设密码。

## 保护规则

公开模式下，一个文件或文件夹只要满足任一条件，就必须隐藏：

- 自身被保护。
- 任一父文件夹被保护。
- 自身包含任一受保护标签。
- 自身位于受保护文件夹下。
- 文件夹因受保护标签被隐藏时，其子内容也隐藏。

父级保护优先：

- 只要父文件夹受保护，子文件和子文件夹无论自身是否保护，都在公开模式下隐藏。
- 子文件不能通过解除自身保护来绕过父文件夹保护。
- 如果文件夹因为标签保护被隐藏，其子内容也在公开模式下隐藏。

标签保护规则：

- 受保护标签在公开模式下不显示。
- 搜索标签时不能搜出受保护标签。
- 添加标签时不能搜出受保护标签。
- 标签自动补全不能出现受保护标签。
- 包含受保护标签的文件和文件夹在公开模式下隐藏。

## 访问入口覆盖范围

公开模式下，所有入口都必须过滤受保护内容，包括但不限于：

- 主文件列表。
- 文件夹浏览。
- 搜索结果。
- 标签列表。
- 标签搜索。
- 添加标签。
- 标签自动补全。
- 收藏列表。
- 最近访问。
- 历史记录。
- 路径跳转。
- 面包屑导航。
- 缩略图展示。
- 文件详情面板。
- 右键菜单。
- 快捷操作。
- 拖拽入口。
- 打开终端入口。
- 复制路径入口。
- 搜索建议。
- 搜索历史。

公开模式不显示隐藏文件数量，也不显示任何“已隐藏内容”的提示。

## 异常访问处理

如果因为意外情况，公开模式下进入了受保护内容：

- 弹窗提示当前内容在公开模式下不可访问。
- 不显示具体受保护文件名或完整路径。
- 用户确认后直接跳转到 `C:\`。
- 如果 `C:\` 不可访问，则跳转到默认安全目录。

建议文案：

> 当前内容在公开模式下不可访问。

## 图标与视觉规则

图标资源：

- 解锁图标：`D:\fileGe\icon\unlock_line.svg`
- 锁定图标：`D:\fileGe\icon\lock_line.svg`

使用规则：

- 公开/隐私模式切换按钮可以使用 unlock / lock 图标。
- 受保护文件左下角显示 lock 图标。
- 受保护文件夹左下角显示 lock 图标。
- lock 图标样式参考右下角 tag 图标。
- 进入受保护文件夹时，面包屑左侧图标由文件夹图标替换为 lock 图标。
- 文件列表中不建议为未保护文件显示 unlock 图标，避免界面噪音。

## 右键菜单

隐私模式下，文件和文件夹右键菜单新增：

- 保护
- 解除保护

位置：

- 放在“收藏/取消收藏”下面。

公开模式下：

- 不显示“保护/解除保护”。
- 不显示当前文件或文件夹是否受保护。
- 不暴露保护相关入口。

## 设置页

设置中新增隐私相关配置：

- 当前模式：公开模式 / 隐私模式。
- 切换模式。
- 修改隐私密码。
- 忘记密码。
- 启动时恢复上次隐私模式。
- 标签保护设置。

设置中的“标签”页：

- 可以设置标签是否保护。
- 仅隐私模式下允许设置标签保护状态。
- 公开模式下不显示受保护标签。
- 公开模式下不显示标签保护开关。

## 搜索规则

公开模式下：

- 搜索返回结果必须过滤保护内容。
- 过滤应覆盖文件名、文件夹名、路径、标签、历史记录和搜索建议。
- 不能只依赖前端隐藏。
- 搜索结果数量不应包含被隐藏内容。
- 搜索高亮和路径提示不能泄露保护内容。

隐私模式下：

- 搜索可以返回受保护内容。
- 受保护内容可以显示 lock 状态。

## 安全边界

该功能目标是本地应用内的隐私隔离，不等同于文件系统级加密。

公开模式下应做到：

- 应用内不可见。
- 应用内不可搜索。
- 应用内不可通过入口访问。
- 不通过标签、数量、历史记录、路径提示泄露保护内容。

如果需要抵御系统级访问、磁盘扫描或管理员权限访问，应另行设计加密方案。
